Set up the extension

Install the OpenAI Codex extension in your IDE and connect it to your account.

Requirements

  • VS Code 1.60+ (or a compatible IDE)
  • An OpenAI API key
  • Internet connectivity

Installation

  1. Open the extensions panel in VS Code.
  2. Search for OpenAI Codex.
  3. Click Install and then Reload if prompted.
Shell 
$ code --install-extension openai.codex
$ code --list-extensions | findstr codex
openai.codex

Authentication

After installation, open the Command Palette and run Codex: Sign in. Paste your API key when prompted.

Settings 
{
  "openai.apiKey": "sk-...",
  "codex.suggestions.enabled": true,
  "codex.inlineCompletions": "on"
}

Next steps

  • Trigger inline completions by typing or pressing Ctrl+Space.
  • Open the Codex panel to view suggestions and examples.
  • Visit the IDE usage guide for advanced tips.
